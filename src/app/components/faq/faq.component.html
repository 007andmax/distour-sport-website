Faq

Как пополнить средства?

  1. Авторизироваться
  2. Зайти в профиль
  3. Выбрать нужное количество монет для пополнения

Как вывести средства?

  1. Авторизироваться
  2. Зайти в профиль
  3. Выбрать нужное количество монет для вывода
  4. Ввести номер карты
  5. Выбрать страну
  6. Нажмите - `Вывести`

Как создать челендж?

  1. Авторизироваться
  2. Зайти список челенджей
  3. Нажать кнопку с иконкой - `+`
  4. Выберите категорию и уровень сложности
  5. Выберите челендж из списка
  6. Нажмите - `Создать`

Как участвовать в челендже?

  1. Авторизироваться
  2. Зайти список челенджей
  3. Выбрать челендж
  4. Нажать - `Участвовать`

Как загрузить видео?

  1. Авторизироваться
  2. Зайти в челендж
  3. Нажать на - `Загрузить`
  4. Выбрать видео, дождаться окончания загрузки`
